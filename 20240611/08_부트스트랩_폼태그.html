08_부트스트랩_폼태그

다양한 폼 사용을 위한 폼 컨트롤 스타일, 레이아웃 옵션, 사용자 정의 컴포넌트의 예시와 사용 가이드입니다.

폼태그 그룹제목

아이디는 8~20글자 사이로 입력

폼입력양식의 크기 조정은 .form-control-lg, .form-control-sm으로 크기 조절 가능

버튼 비활성화 - disabled

disabled 속성을 추가하면 비활성화 외형으로 변경되고 포커싱되지 않는다

읽기전용 - readonly

readonly 속성을 추가함으로써 입력 값의 변경을 막을 수 있습니다. disabled와 달리 readonly 입력값은 계속 포커싱 및 선택이 가능

라디오버튼 - label태그에 .form-check-label, input 태그에 .form-check-input

체크박스

선택목록 - select, option

select 태그에 .form-select 서식을 적용하고, 크기 조정시는 .form-select-lg 또는 .form-select-sm 서식을 적용

select태그에 multiple 속성을 넣으면 다중 선택도 가능. size 속성으로 크기도 조정 가능.

다중텍스트 박스

.form-control일때는 가로 100%, form-control-lg, form-control-sm을 적용할 수 있음.

높이를 적용하고 싶으면 css 서식으로 조절 가능
