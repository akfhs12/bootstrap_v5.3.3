작동원리

  • 내비게이션 바는 .navbar를 .navbar-expand[-sm|-md|-lg|-xl|-xxl]로 감싸야 하며, color scheme 클래스가 필요하다
  • 내비게이션과 그 콘텐츠는 기본적으로 유동적. container를 변경하여 다양한 방법으로 가로폭을 제한할 수 있다.
  • 내비게이션에서 배치 및 간격 제어는 spacing의 flex를 사용
  • 내비게이션은 기본적으로 반응형이지만 간단히 변경이 가능. 반응형 동작은 collaps javascript 플로그인에 의존
  • 접근성을 확보하려면 nav 요소를 사용하거나 혹은 div와 같이 보다 반응적인 요소를 사용하는 경우에 role="navigation"을 추가하여 스크린 리더 사용자를 위한 명시를 하여야 함
  • 현재 페이지는 aria-current="page"를 세트 안에서의 현재 아이템에는 aria-current="true"를 사용하며, 현재 아이템을 표시함
  • .navbar 기본 클래스로 범위가 지정된 css 변수를 사용하여 테마를 지정할 수 있다
  • navbar-light는 더 이상 사용되지 않으며 추가 스타일을 추가하는 대신 css 변수를 재정의하도록 navbar-dark가 재작성됨
